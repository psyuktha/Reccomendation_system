# Movie Recommendation System

## Overview

This project is a movie recommendation system that utilizes two approaches:
1. **Collaborative Filtering**: Based on user reviews and KNN cosine similarity.
2. **Content-Based Filtering**: Based on the genre of the movies using TruncatedSVD and cosine similarity.

The dataset used for this project is the MovieLens dataset available on Kaggle.

## Dataset

The dataset can be downloaded from [MovieLens Dataset on Kaggle](https://www.kaggle.com/datasets/shubhammehta21/movie-lens-small-latest-dataset).

### Dataset Structure
- **ratings.csv**: Contains user ratings for movies.
- **movies.csv**: Contains movie information including genres.
